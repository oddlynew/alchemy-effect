# Provider Catalogs

Provider catalog notes are working documents for resource-factory waves. They are not generated API
docs and they are not required for ordinary package changes.

When a factory wave catalogs a provider service, place its notes under:

```text
docs/provider-catalog/<Cloud>/catalog/<service>.md
```

Use these files to track resource coverage, lifecycle operation mapping, testability, entitlement
gates, and SDK patches discovered while live-testing Alchemy resources against Distilled providers.
